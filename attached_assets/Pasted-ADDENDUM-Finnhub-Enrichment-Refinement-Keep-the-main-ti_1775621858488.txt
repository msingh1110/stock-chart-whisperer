1. Main ticker page: keep existing layout

Do not redesign the main ticker page.
Keep the current ticker detail layout, signal block, probabilities, charts, explanation, and existing market data as they are.

Only add these two enhancements:

A. Fundamentals snapshot
Add a compact “Fundamentals Snapshot” section on the main ticker page with a small set of high-value fields from Finnhub.

Include only the most useful fields that are available:
- market cap
- P/E ratio
- EPS
- 52-week high
- 52-week low
- beta
- shares outstanding

Rules:
- Show only fields that are available
- Keep the section compact and visually clean
- Use a simple grid or two-column layout
- Do not overload the page with too many financial metrics

B. Optional news section
Add a “Latest News” section on the main ticker page using Finnhub news.

Include:
- latest 3 headlines
- source
- published time
- link to article

Optional:
- lightweight sentiment tag if already implemented
- otherwise just show headlines cleanly

Rules:
- Keep this section below the main ticker content
- Keep it compact and readable
- Open news links in a new tab
- If no news is available, hide the section gracefully
